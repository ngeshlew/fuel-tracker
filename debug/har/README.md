# HAR Files Directory

This directory contains HTTP Archive (HAR) files for debugging network requests and API calls.

## Usage

Place your HAR files here for:
- Debugging API requests/responses
- Analyzing network performance
- Troubleshooting API issues
- Testing API endpoints

## File Naming Convention

Suggested naming: `[domain-or-service]-[date].har`

Examples:
- `fuel-tracker-up.netlify.app-2025-11-21.har`
- `fuel-tracker.up.railway.app-2025-11-21.har`
- `api-debug-2025-11-21.har`
